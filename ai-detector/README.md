## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
  
## Usage

1. Enter or paste the text you want to analyze in the text area
2. Click "Analyze Text"
3. View the results:
   - Overall prediction percentage
   - Confidence bar
   - Highlighted sentences (AI-generated content is highlighted)
   - Detailed analysis
